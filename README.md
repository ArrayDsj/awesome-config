# awesome-config
a common useful server config.
## 1. [nginx](nginx)

- nginx.conf:总的配置文件
- proxy.conf：反向代理相关的配置文件
- gzip.conf：gzip相关
- uuid4.lua：生成uuid的lua脚本

## 2. [tomcat](tomcat)

- java_opts.conf：jvm参数配置
- connector.conf：tomcat connector参数配置

## 3. [mysql](mysql)

- my.cnf: mysql相关配置

## 4. [redis](redis)

- redis.conf: redis相关配置
